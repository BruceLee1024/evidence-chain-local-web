## 项目概述

面向**施工单位商务结算人员**的证据链管理工具。解决施工结算中「证据找不到、资料混乱、对账扯皮」三大痛点。覆盖签证变更单、隐蔽工程记录、材料进场记录、月度计量记录四类核心证据的全生命周期管理。

### 功能范围

```
证据链管理系统 v1.0
├── 📂 项目主页（Dashboard — 证据概览 + 最近记录）
├── 📋 证据管理（4 类证据 CRUD：签证变更单 / 隐蔽工程 / 材料进场 / 月度计量）
├── 🔍 全局搜索（关键词 + 类型 + 日期筛选）
├── 📦 结算组卷（导入 Excel → 规则匹配 → 人工校对 → ZIP 导出）
└── ⚙️ 基础设置（项目管理 + 部位管理）
```

### 核心设计决策

- **不做大而全**：只聚焦 4 类结算高争议证据，不做通用文档管理
- **签证单只录入归档**：不走线上审批流，签字走线下
- **结算匹配分三层**：精确匹配（部位完全一致）→ 模糊匹配（关键词包含）→ 时间关联
- **本地优先**：SQLite 本地存储 + 文件归档，数据完全自控
- **v1.0 暂不支持编辑和删除**：只做新增和查看

### 数据模型

```
projects → locations
projects → evidence（4 种 type：variation / hidden / material / monthly）
  evidence → files（via evidence_files，带 role 分类）
projects → settlement_sessions → settlement_items → settlement_links → evidence
```
